# PHP CRUD Project

This project is a simple PHP CRUD application that allows you to insert, display, update, and delete records from a MySQL database.

## Features
- Insert users
- Display all users
- Update user information
- Delete user records

## Database

The database file `classa.sql` is included in the project folder. It contains the table structure and sample data.

## How to Run
1. Copy the project folder to your `htdocs` (XAMPP).  
2. Import `classa.sql` in phpMyAdmin.  
3. Open `display.php` in the browser: `http://localhost/PATIENT SYSTEM/display.php`.
